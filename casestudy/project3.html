Project 3: Interactive Piano Toy

Web Toy

Click to visit project website

In this project, I designed a interactive piano toy using javascript. User can play this piano either by cliking directly on the piano key or pressing the keyboard. The image of a piano drawing on the top is also clikable, and will play song automatically when it is being clicked.

To create this piano, I first recorded the sound of 36 notes seperately, and linked each file to the corresponding piano key, and then created events through javascript: when a piano key is clicked or the corresponding keyboard is pressed, the audio file linked to the key will be played.

Audio files

Javascript

I visited some websites where we can play piano online, and I found that they can't simulate the real feeling of playing the piano, that is, the touch of pressing the keys, because those pianos can only make sound by clicking with the mouse. Therefore, I had this idea of creating a piano that can be played through pressing the keyboard.
